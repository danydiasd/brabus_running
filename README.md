Página de divulgação simples e visualmente alinhada para o 2º Treinão do Brabus Run Club usando os dados do formulário informado.
Aplicando a  identidade visual em vermelho, preto e branco como solicitado pela equipe da academia, para reforçar a marca e aumentar conversões para inscrições.


Há apenas o arquivo estático index.html contendo layout responsivo com CSS embutido e variáveis de cor (--red, --bg, --card, etc.).
Estrutura de página em seções solicitadas: 1. Quem somos, 2. Objetivo da corrida e 3. Descrição da corrida com detalhes de data, percurso, horários e local.
Há a lista de benefícios/ativações (degustação, whey, pré-treino, fotógrafo, coffee break, sorteio) e seção explicando por que preencher o formulário.
